Description of Figure CertificationPath

Link to figure Certification Path.

This figure consists of three boxes arranged horizontally:

  • The first box is divided into three sections horizontally. These sections contain the following text, from top to bottom:

    • Issuer = CA2
    • Subject = Alice
    • Alice's Public Key

    The last section also contains an image of a key.

  • The second box is divided into three sections horizontally. These sections contain the following text, from top to bottom:

    • Issuer = CA1
    • Subject = CA2
    • CA2's Public Key

    The last section also contains an image of a key.

  • The third box contains the text "CA1's Public Key" and an image of a key.

Arrows connect the following boxes and sections

  • From "CA1's Public Key" to "Issuer = CA1"
  • From "Subject = CA2" to "Issuer = CA2"
