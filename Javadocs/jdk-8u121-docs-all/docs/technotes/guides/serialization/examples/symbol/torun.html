Running the Example

Complete the following procedure to compile and run the example:

  1. From yourbasedir/objectserialization/ReplaceableInterface, compile the file Substitute.java using JDK™ 1.2 or higher: 
      javac Substitute.java
  2. Run the compiled file: 
      java Substitute
