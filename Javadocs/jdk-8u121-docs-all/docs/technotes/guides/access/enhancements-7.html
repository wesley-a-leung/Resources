Accessibility Enhancements in Java SE 7

Java Accessibility Guide > Java Accessibility API (JAAPI) Programmer's Guide > Accessibility Enhancements in Java SE 7

Java Access Bridge is included with Java SE Runtime Environment (JRE) Release 7 Update 6 (7u6) and later. Java Access Bridge 2.0.3 is included with JRE 7u6. By default, Java Access Bridge is not enabled. See Enabling and Testing Java Access Bridge for more information. See also RFE 7128768.
