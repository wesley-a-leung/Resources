Changes and New Features

Changes and New Features in Java SE 5.0

  • The management interface for the logging facility is implemented. Refer to Monitoring and Management API for the Logging Facility.
  • Handlers can now be added to Loggers using the logging configuration file. See the .handlers property in java.util.logging.LogManager. See 4635817 for details.
