This diagram shows features associated with four deployment types: standalone applications; self-contained applications; Java Web Start applications; and applications embedded in a web page.

Eight features are displayed, which span one or more columns. The features are named here and described in the following sections of the document.

Feature 1: Preloader: Applies to all deployment types, JavaFX applications only.

Feature 2: Built-in proxy support: Applies to self-contained applications, Java Web Start applications, and applications embedded in a web page.

Feature 3: Desktop integration via shortcut or menu. Applies to self-contained applications, Java Web Start applications, and applications embedded in a web page.

Feature 4: Auto-updates: Applies to Java Web Start applications and applications embedded in a web page, also to applications that were earlier downloaded from the web using either of these execution modes.

Feature 5. Fine-tune user experience with Deployment Toolkit: Applies to Java Web Start applications and applications embedded in a web page.

Feature 6: Communicate via JavaScript: Applies to applications embedded in a web page.

Feature 7: No system installation of Java required: Applies to self-contained applications.
