Preface

The Java SE Installation Guide describes how to install the Java Development Kit (JDK), Java Runtime Environment (JRE), and server JRE on Oracle Solaris, Linux, Microsoft Windows, and OS X computers.

Audience

This document is intended for users who are installing any variant of the Java SE platform.
