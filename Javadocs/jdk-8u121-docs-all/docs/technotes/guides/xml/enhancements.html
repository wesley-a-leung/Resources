Enhancements in Java SE 7u45

Three processing limits have been added to JAXP in the 7u45 release. For more information, see the new Processing Limits lesson in the Java Tutorial.

Enhancements in Java SE 7u40

The 7u40 release contains Java API for XML Processing (JAXP) 1.5, which adds the ability to restrict the set of network protocols that may be used to fetch external resources. For more information, see JEP 185: JAXP 1.5: Restrict Fetching of External Resources and the new JAXP 1.5 and New Properties lesson in the Java Tutorial.

Enhancements in Java SE 7

JAXP

  • The Java SE 7 release now contains Java API for XML Processing (JAXP) 1.4.5.
  • There have been many bug fixes and some improvements over the previous version, particularly in the areas of conformance, security, and performance.
  • The specification is still JAXP 1.4, however StAX has been upgraded to StAX 1.2 in accordance with the JSR 173 Maintenance Review 3.
For more information, see the JAXP 1.4.5 Release Notes and the JAXP 1.4.5 Change log.

JAXB

The Java SE 7 release supports Java Architecture for XML Binding (JAXB) 2.2.3. For more information, see the JAXB Change log for JAXB RI 2.2 and above.

JAX-WS

The Java SE 7 release supports Java API for XML Web Services (JAX-WS) 2.2.4. For more information, see the JAX-WS Change log for JAX-WS RI 2.2 and above.
